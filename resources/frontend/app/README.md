# app vue + auth + multilanguage

## deploy
```
yarn install
```

### compiles and hot-reloads for development
```
yarn run serve
```

### compiles and minifies for production
```
yarn run build
```

### run tests
```
yarn run test
```

### run unit tests
```
yarn run test:unit
```
